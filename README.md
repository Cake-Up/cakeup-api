# cakeuperp-rest-api-jwt
CakeUpERP de uma rest api usando autenticação via JWT, clean architecture e tests automatizados


## Tecnologias utilizadas
* .NET 6
* C#
* Xunit
* Linq
* MySql

## Como Instalar?
1. Clone este projeto.
```bash
git clone https://github.com/oleandrosantos/cakeuperp-rest-api-jwt
```

2. Encontre o diretorio no seu terminal e instale o pacote atravez do comando abaixo:
```bash
dotnet new install .\
```

3. Inicie o projeto atravez do comando abaixo, substituia o NomeDoProjeto pelo nome desejado.
```bash
dotnet new rest-api-jwt -o NomeDoProjeto
```
